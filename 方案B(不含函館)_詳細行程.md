## Day 2｜8/1（六）⚾ ES CON Field 看球賽

## Day 4｜8/3（一）🚗 租車出發｜札幌 → 支笏湖 → 洞爺湖

**租車：** 札幌市區取車（建議 Toyota Rent a Car 或 Nissan Rent a Car，記得含兒童座椅）  
**行車：** 札幌 → 支笏湖約 40 分 / 支笏湖 → 洞爺湖約 65 分

| 時間 | 行程 |
|------|------|
| 08:30 | 飯店退房，前往租車公司取車 |
| 09:30 | 🛣️ 出發，往支笏湖（走道東自動車道・支笏湖方向）|
| 10:15 | 抵達 **支笏湖**（支笏洞爺國家公園） |
| 10:30 | 🛶 **獨木舟體驗**（預約制，建議提前上網預約）推薦業者：支笏湖ネイチャーセンター。約 1.5-2h，¥5,000-7,000/人。湖水透明度極高，天氣好可見湖底！|
| 12:30 | 🐟 **水中步道**（水中遊覧船）｜坐玻璃底船看水中魚類與植物，小孩超感興趣，¥1,500/大人  |
| 13:30 | 🍱 **午餐**｜支笏湖溫泉街小餐廳，推薦「支笏の 四季」鮭魚丼 |
| 14:30 | 前往洞爺湖（約 65 分）|
| 16:00 | 抵達洞爺湖，飯店 Check-in |
| 17:30 | 🚢 **洞爺湖遊覧船**（最後一班約 17:00-18:00，看季節）若趕得及可今天先搭，約 50 分，¥1,500/大人 |
| 20:45 | 🎆 **洞爺湖湖畔煙火大會**（每年 4 月下旬到 10 月下旬每晚施放，約 20:45，持續 20 分鐘！）⭐ 千萬別錯過！ |

**住宿：** 洞爺湖溫泉  
推薦：湖畔亭 / 望羊蹄 / Toya Manseikaku（溫泉旅館，有湖景，小孩游泳池）

**備註：**  
- 支笏湖獨木舟強烈建議提前在官網預約（夏季滿員快）  
- 洞爺湖煙火每晚固定施放，站在飯店湖畔即可欣賞  
- 當晚溫泉旅館通常附晚餐（海鮮懷石），記得確認是否含餐

## Day 5｜8/4（二）洞爺湖｜火山全日遊

**交通：** 開車（洞爺湖周邊）

| 時間 | 行程 |
|------|------|
| 07:30 | 旅館早餐（通常溫泉旅館附早餐）|
| 09:00 | 🚢 **洞爺湖遊覧船**（若昨晚沒搭）｜中島上有蝦夷鹿，船上風景優美 |
| 10:00 | 🌋 **昭和新山**（開車約 20 分）｜1944年火山噴發形成的小山丘，熊牧場在旁 |
| 10:30 | 🐻 **昭和新山熊牧場**（可選）｜小孩可近距離餵食棕熊，¥1,500/大人 |
| 11:30 | 🚡 **有珠山纜車**｜搭纜車上火山口展望台（¥1,800 來回/大人），俯瞰洞爺湖絕景 |
| 12:30 | 午餐｜昭和新山周邊小食堂 |
| 14:00 | 🏛️ **洞爺湖火山科學館（Volcano Science Museum）**｜互動展示，了解 2000 年有珠山噴火歷史，小孩很感興趣 |
| 15:30 | 洞爺湖湖畔散步・溫泉老街逛街 |
| 17:30 | 返回旅館，泡溫泉放鬆 |
| 20:45 | 🎆 **洞爺湖煙火**（第二晚！）|

**住宿：** 同 Day 4，洞爺湖溫泉

**備註：**  
- 有珠山纜車下方可看 2000 年噴火後留下的廢墟建物（直接被熔岩掩埋的廢校），非常震撼  
- 天氣好時有珠山頂視野極佳，可遠眺羊蹄山
